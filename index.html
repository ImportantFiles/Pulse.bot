How to Use Pulse.bot

📘 Calculation Reference

Quick reference for the values Pulse.bot calculates rather than reads directly from the screenshot.

Growth %

Net Deposit = Total Deposit − Total Withdrawal

Growth % = ((Current Balance − Net Deposit) ÷ Net Deposit) × 100

Shows the account's overall gain or loss based on the client's Net Deposit, taking withdrawals into account before calculating the percentage return.

Floating P/L

Floating P/L = Equity − Current Balance

Represents the unrealized profit or loss from currently open trades.

Track Record

Track Record = First Closed Trade Date → Today's Date

Represents the total number of days the trading account has been actively trading.

OCR Extraction

These values are extracted directly from the STT Account Details screenshot and validated before any calculations are performed:

  • Current Balance
  • Equity
  • Closed Profit
  • Total Deposit
  • Total Withdrawal

Important Notes

  • Growth is calculated and is no longer read from the screenshot.
  • Net Deposit is calculated as Total Deposit − Total Withdrawal.
  • Floating P/L is always calculated using Equity and Current Balance.
  • Track Record is calculated using the First Closed Trade Date entered by the user.
  • Positive and negative values are preserved during OCR and calculations.
  • All percentages are rounded to two decimal places.
  • Currency values retain proper formatting.
  • If a required value can't be extracted or is invalid, Pulse.bot displays a clear warning instead of generating an incorrect calculation.
  1. Open STT and go to the Account Dashboard page.
  2. Take a screenshot of the Account Details section and half of the Account Info section from STT. View reference screenshot
  3. Upload the screenshot using any of the following:
    • Drag & Drop
    • Manual Upload
    • Ctrl + V (Paste)
  4. Wait for Pulse.bot to finish reading the screenshot.
  5. Select the correct trading system and enter the date of the first closed trade.
  6. Example: 2/18/2025
  7. Click "Add Entry".

If this is the only trading system, click:

Generate Summary

Then click:

Copy to GHL

Finally, paste the generated summary directly into the client's GHL Notes.

For Multiple Systems

Repeat the same process for every trading system.

Do NOT click Generate Summary until ALL systems have been added to the list.

Once every system has been added:

  • Click Generate Summary
  • Click Copy to GHL
  • Paste the generated summary directly into GHL Notes.
